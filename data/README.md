# Documentation Folder

This folder contains human-readable documentation for the FinGuard AI project.

## What goes here
- `evaluation_report.md`
- methodology notes
- architecture overview
- experiment summaries
- diagrams and figures used in reports

## Purpose
This folder explains how the fraud detection system works, how the model was evaluated, and what limitations or future improvements exist.

## Suggested files
- `evaluation_report.md`
- `methodology.md`
- `architecture.md`
- `figures/`

## Notes
- Keep this folder focused on explanation, not code.
- Write in clear project-report style so it can support a thesis or GitHub portfolio presentation.
